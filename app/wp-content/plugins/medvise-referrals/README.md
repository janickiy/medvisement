# Алгоритм работы:

<ol>

<li>
В каждой статье на сайте отображаются кнопки ссылки на статью.
Сами ссылки формируются из ID статьи и ID текущего пользователя.
Затем каждой цифре случайно присваивается буква по заданному маппингу.<br><br>
Первые два пункта сделаны в рамках темы carenow\inc\referral.php
</li>

<br>

<li>
После открытия ссылки, происходит редирект на сайт через POST форму
(передается ID аффилята, статьи и открывается статья через редирект)<br>
На случай, если пользователь не авторизован - сначала выставляется кука с ID аффилята
на 7 дней.<br>
Если пользователь авторизован - читаем куку и выставляем meta пользователю с 
ID аффилята и временем посещения
(на момент логина, т.е. может пройти 7 дней до момента авторизации 
+ 7 дней после авторизаци)<br><br>

Также здесь происходит первичная проверка на наличие оплаченных заказов перед привязкой
к аффиляту.
</li>

<br>

<li>
В заказ записывается мета '_yith_wcaf_referral'

- при оформлении заказа пользователем через корзину
- при продлении подписки (при условии, что у изначального заказа стоит аффилят)
- при оформлении заказа из админки (если указано руками в метабоксе)
</li>

</ol>

## Статусы
<ol>
<li>Не подтверждено ('not-confirmed') - Комиссия задана заказу, но заказ не оплачен</li>
<br>
<li>В ожидании ('pending') - Оплата по заказу прошла. Комиссия подтверждена.</li>
<br>
<li>Ожидает выплату ('pending-payment') - Комиссия переведена на баланс и ожидает выплаты.</li>
<br>
<li>Выплачено ('paid') - Выплата была произведена по комиссии</li>
<br><br>
<li>Отменено ('cancelled') - Отменено, хз</li>

<li>Возмещено ('refunded') - Заказ возмещен. 
<li>Может быть возмещен и после выплаты. Надо следить.</li>

<li>В корзине ('trash') - В мусор кинули комиссию</li>
</ol>

### Получается алгоритм работы:
1) Прокинул ожидающие комиссии в статус "В ожидании"
2) Зашел в список партнеров, увидел нужного - там баланс висит и кнопка "Оплатить сейчас"
3) Создается выплата на странице "выплаты" с суммированными комиссиями
4) Заходишь на выплаты и жмешь сменить статус на "Завершено". Тут ты типа уже бабки перевел человку
